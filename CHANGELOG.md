# Changelog

All notable changes to skill-habit are documented here.
Format: [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)

## [0.0.1] - 2026-06-28
### Added
- Initial release: tracks slash-command usage across Claude Code sessions
- Frequency-ranked shortcut skill generation (numeric and command modes)
- Management UI with heatmap, top skills, hourly/weekday distribution charts
- Association prediction: surfaces skills you typically chain next
- Plugin skill support (e.g. voicemode:converse)
- Configurable prefix, blacklist, time windows, theme (light/dark/system)
